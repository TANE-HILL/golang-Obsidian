# Массив
Это последовательность [[1.1. Переменные|элементов]] [[1.2. Типы данных|одного типа]] фиксированной длины(коробка "фиксированного размера", в которой хранятся элементы одного типа.).

![[Pasted image 20260717142148.png|422]]

В [[1.0. Go Basics|Go]] массивы имеют **жёсткий размер**, который задаётся при создании.
	`[3]int` — это всегда ровно 3 целых числа.

#### Примеры:
```go
var arr1 [3]int        // массив из 3 целых чисел
var arr2 [5]string     // массив из 5 строк
var arr3 [2]bool       // массив из 2 булевых значений
```

#### Если мы не заполняем массив сразу, то все его [[1.1. Переменные|элементы]] получают **нулевое значение**:
- `int` → `0`
- `string` → `""` (пустая строка)
- `bool` → `false`

**Нулевые значения при создании массива:**
```go
var arr [3]int
fmt.Println(arr)
```
**Вывод:**
```
[0 0 0]
```

# Особенности массивов:
- **<u>Индексация начинается с 0.</u>**
- Если элемент не инициализировать, он будет иметь **нулевое значение** (`0`, `false`, `""` - в зависимости от типа).
- Размер массива — часть его типа (<u><font color="#f79646">[3]int</font> и <font color="#4bacc6">[4]int</font> — разные типы</u>).
- Попытка <u>обратиться за пределы</u> → `panic: index out of range`.

---

**Что значит “инициализировать”?**

Инициализация — это когда ты **присваиваешь значение переменной** (или элементу массива) **при создании**.

Если этого не сделать, Go автоматически заполняет такие элементы **значением по умолчанию** для их типа.
