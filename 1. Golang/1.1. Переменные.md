# Cоздание переменной в [[1.0. Go Basics|Go]]:
### 1. Полная форма переменной:
```go
var age int = 25
var name string = "Alex"
```
`var` → *«создай переменную»*
`age`, `name` → *имена*
`int`, `string` → *типы*
`25`, `"Alex"` → *значения*
### 2. Краткая форма переменной:
```go
age := 25
name := "Alex"
```

**Go сам понимает тип переменной по значению:**
	*"25" автоматически определяется как int*
	*"10.5" - как float*
	*[!] Если нужен float, но число целое, можно написать:*
	a := 10. *(с точкой в конце)*
	*Или использовать способ 3.*
### 3. Краткая форма с принудительным указанием [[1.2. Типы данных|типа данных]]:
```go
age := float64(12)
```
### 4. Объявление нескольких переменных:
```go
a, b, c := 1, 2, 3
```

# Имена переменных в [[1.0. Go Basics|Go]]:
- Имя должно начинаться с буквы или **`_`**;
- Дальше можно буквы и цифры;
- Принято писать в стиле **camelCase**: `userName`, `isAdmin`.
