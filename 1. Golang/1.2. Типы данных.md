[[1.0. Go Basics|Go]] — **строго типизированный** язык.
У каждой [[1.1. Переменные|переменной]] всегда есть определённый **тип**, и этот тип нельзя перепутать.

**Например**: число нельзя превратить в строку без явного преобразования, а в булевую переменную нельзя записать текст.

# Основные типы переменных в Go

## 1. Целые числа ~ <font color="#ff0000">int</font> ~

| Тип       | Минимум                    | Максимум                  | Биты |
| --------- | -------------------------- | ------------------------- | ---- |
| **int8**  | -128                       | 127                       | 8    |
| **int16** | -32 768                    | 32 767                    | 16   |
| **int32** | -2 147 483 648             | 2 147 483 647             | 32   |
| **int64** | -9 223 372 036 854 775 808 | 9 223 372 036 854 775 807 | 64   |
- Кроме `int` есть «узкие» типы: `int8`, `int16`, `int32`, `int64`. Они отличаются количеством памяти и диапазоном значений.
    *Например, `int8` хранит числа от -128 до 127.*

Обычно в реальных программах используют просто `int`, а маленькие типы (`int8` и т.п.) нужны, если вы работаете с файлами, сетевыми протоколами или оптимизируете память.

## 2. Дробные числа ~ <font color="#ff0000">float64</font>, <font color="#ff0000">float32</font> ~

- `float64` — основной тип для дробей.
- `float32` занимает меньше памяти, но используется редко (*в основном в графике или научных расчётах*).

⚠️ Важно: числа с плавающей точкой могут храниться не точно. Это особенность компьютеров.
*Например, 0.1 может храниться как 0.10000000000000001.* 

## 3. Строки ~ <font color="#ff0000">string</font> ~

- Строка в Go — набор символов в **кодировке <font color="#f79646">UTF-8</font>**.
- Пишется в кавычках `"..."`.
```go
name := "Alex"
fmt.Println("Name:", name)
```

## Что такое <font color="#f79646">UTF-8</font> ?

Компьютер хранит **числа**, называемые **кодами символов**.

Например, символ `'A'` имеет код **65**, а `'П'` — **1055**.

**UTF-8** — это способ (формат) хранить эти коды в памяти.

Каждый символ занимает от **1 до 4 байт**:

- английские буквы — 1 байт,
- русские и другие — 2 или больше байт,
- эмодзи 😊 — 4 байта.

Это значит, что строка `"Привет"` занимает **12 байт**, хотя состоит из **6 символов**.

---

Можно складывать строки (`"Hello, " + name`) или измерять их длину функцией `len()`.

Но  `len()` возвращает **количество байт**, а не символов:

```go
word := "Привет"
fmt.Println(len(word)) // 12, а не 6 (потому что каждая буква = 2 байта)
```

Если нужно узнать именно количество **символов**, можно преобразовать строку в срез `rune`:
```go
fmt.Println(len([]rune(word))) // 6
```

## 4. Булевы значения ~ <font color="#ff0000">bool</font> ~

- Логический тип: только два значения — `true` или `false`.
- Используется для условий и логики.

Булевы переменные можно задавать из выражения:
```go
age := 20
isAdult := age >= 18
fmt.Println(isAdult) // true
```
Либо так:
```go
isAdult := true
isStudent := false
```

## 5. Беззнаковые числа ~ <font color="#ff0000">uint</font> ~

<font color="#ff0000">uint</font> (или «<font color="#ff0000">Unsigned int</font>») — числа **только положительные** (нельзя хранить минус).
	Постоянно используются на практике. Когда заранее известно, что значение не может быть отрицательным — например, длина массива, количество элементов, возраст, ID и т.д.
Можно задать переменную так:
```go
var items uint = 10
```
Или так:
```go
items := uint(10)
```

## 6. Символы ~ <font color="#ff0000">rune</font> и <font color="#ff0000">byte</font> ~

В [[1.0. Go Basics|Go]] нет отдельного типа **«символ»**, как в некоторых других языках.

На самом деле **каждый символ — это число**, представляющее его **код** в таблице символов.

Каждая буква, цифра или знак хранится как **число** (код символа).

**Например:**
- **<font color="#ffff00">A</font>** → код **<font color="#ffff00">65</font>**
- **<font color="#ffff00">B</font>** → **<font color="#ffff00">66</font>**
- **<font color="#ffff00">П</font>** → **<font color="#ffff00">1055</font>**

Код каждого символа в [[1.0. Go Basics|Go]] определяется таблицей символов **<font color="#ffff00">Unicode</font>** — это огромный стандарт, где каждому символу в мире присвоен свой числовой код.
![[Pasted image 20260715092747.png|441]]
### <font color="#ffff00">ASCII</font> и <font color="#ffff00">Unicode</font> — в чём разница

- **<font color="#ffff00">ASCII</font>** — старая таблица, в ней есть только английские буквы, цифры и символы (`A-Z`, `0-9`, `!@#` и т. д.).
    Каждый символ занимает **1 байт** (число от 0 до 127).

- **<font color="#ffff00">Unicode</font>** — современный стандарт, где есть **все языки мира** и даже эмодзи 😊.
    Каждый символ может занимать **несколько байт**.

### <font color="#ff0000">byte</font> и <font color="#ff0000">rune</font> в [[1.0. Go Basics|Go]]

- **`byte`** — это просто другое имя для типа `uint8`.
    Используется, когда нужно работать **с байтами** (например, с файлами или сетевыми данными).

- **`rune`** — это `int32`.
    Используется, когда нужно работать **с символами** (в кодировке Unicode).

### Примеры
```go
letter := 'A' // rune
fmt.Println(letter)       // 65 (числовой код символа)
fmt.Printf("%c\n", letter) // A (формат %c выводит символ)
```
Кавычки `'A'` — это не строка, а **символ** (`rune`).

Если взять строку `"Привет"`, то она состоит из нескольких символов (rune), но каждый символ занимает **по несколько байт.

```go
word := "Привет"
fmt.Println([]byte(word)) // байты
fmt.Println([]rune(word)) // коды символов (rune)
```
Результат будет примерно такой:
```
[208 159 209 128 208 184 208 178 208 181 209 130]
[1055 1088 1080 1074 1077 1090]
```
Первый массив — это **байты**
Второй — **коды символов** в Unicode.

Таким образом:

| Тип    | Что хранит                      | Когда использовать                                 |
| ------ | ------------------------------- | -------------------------------------------------- |
| `byte` | 1 байт (обычно ASCII)           | работа с файлами, сетью, низкоуровневые операции   |
| `rune` | код символа в Unicode (`int32`) | работа с текстом, русскими буквами, эмодзи и т. д. |
## Значения [[1.1. Переменные|переменных]] по умолчанию

```go
var x int // x = 0
var x float64 // x = 0.0
var s string // s = ""
var b bool // b = false
```

## Константы

<font color="#ff0000">const</font> - значения, которые никогда не меняются.
```go
const Pi = 3.1415
const AppName = "Task Manager"
```
Если нужно объявить несколько констант подряд:
```go
const (
    StatusOK       = 200
    StatusNotFound = 404
    StatusError    = 500
)
```
	Это удобно, когда значения логически связаны.

## Выражения и операции

**Арифметика:**
```go
a := 10
b := 3

sum := a + b   // 13
diff := a - b  // 7
mul := a * b   // 30
div := a / b  // 3 (целочисленное деление)
mod := a % b   // 1 (остаток от деления)
```

**Строки:**
```go
first := "Go"
second := "lang"
result := first + second  // "Golang"
```

**Логика:**
```go
age := 20
isAdult := age >= 18   // true
```

Когда нужно красиво подставить значения:
