# Зачем нужны условия и циклы

Когда мы пишем программы, почти всегда нужно:
- принять решение (*если пользователь ввёл правильный пароль — войти, иначе — отказать*),
- повторить действие много раз (*вывести все элементы списка», «сосчитать от 1 до 100*).

Для этого есть:
- **условия (`if`, `else`)** — отвечают на вопрос «да/нет» и управляют выбором пути;
- **циклы (`for`)** — повторяют действия до тех пор, пока выполняется условие;
- **логические выражения** **(`&&`, `||`, `!`)** — помогают комбинировать условия.

# Условия в [[1.0. Go Basics|Go]] ~ <font color="#ff0000">if</font>, <font color="#ff0000">else</font> ~

### Пример условия ( <font color="#ff0000">if</font> ):
```go
age := 20
if age >= 18 {
    fmt.Println("You are an adult")
}
```
	- Проверяем выражение `age >= 18`.
	- Оно возвращает либо `true`, либо `false`.
	- Если `true` → выполняется код внутри фигурных скобок.
	- Если `false` → этот блок просто пропускается.
### Вывод:
```Run
You are an adult
```

### Пример условия ( <font color="#ff0000">else</font> ):
```go
if age >= 18 {
    fmt.Println("You are an adult")
} else {
    fmt.Println("You are a minor")
}
```
	- Если условие `age >= 18` истинно → выполняется первая ветка.
	- Если ложно → выполняется вторая ветка (`else`).
### Вывод:
```
You are an adult
```

### Пример нескольких условий (<font color="#ff0000">else if</font>):
```go
age := 15
if age < 7 {
    fmt.Println("Kindergarten")
} else if age < 18 {
    fmt.Println("School")
} else if age < 23 {
    fmt.Println("University")
} else {
    fmt.Println("Work")
}
```
	- Если возраст меньше 7 → «Kindergarten».
	- Иначе, если меньше 18 → «School».
	- Иначе, если меньше 23 → «University».
	- Если ни одно условие не подошло → выполняется блок `else`.
#### Вывод (при **age := 15**):
```
School
```

---

**⚠️ Условия проверяются по порядку сверху вниз, как только одно выполнилось — остальные игнорируются.**

**⚠️ Фигурные скобки { } обязательны.**

# Логические операторы ~ <font color="#ff0000">И</font>, <font color="#ff0000">ИЛИ</font>, <font color="#ff0000">НЕ</font> ~
Нужны для проверки нескольких условий сразу.

### Логический оператор " <font color="#ff0000">И</font> "   ~ <font color="#ff0000">&&</font> ~
```go
age := 20
hasLicense := true
if age >= 18 && hasLicense {
    fmt.Println("You can drive")
}
```
	- Возраст >= 18 И есть права.

### Логический оператор " <font color="#ff0000">ИЛИ</font> "   ~ <font color="#ff0000">||</font> ~
```go
age := 16
hasPermission := true
if age >= 18 || hasPermission {
    fmt.Println("Access granted")
} else {
    fmt.Println("Access denied")
}
```
	Здесь доступ дают либо по возрасту, либо по разрешению. Достаточно одного условия.

### Логический оператор " <font color="#ff0000">НЕ</font> "   ~ <font color="#ff0000">!</font> ~
```go
isRaining := false
if !isRaining {
    fmt.Println("Go for a walk")
}
```
	`!isRaining` превращает `false` в `true`. То есть условие читается как «если НЕ идёт дождь».

# Конструкция ~ <font color="#ff0000">Switch</font> ~

Иногда нужно проверить <mark style="background:rgba(240, 200, 0, 0.2)"><font color="#ffff00">одну переменную на несколько значений</font></mark>.
С помощью <font color="#ff0000">if else</font> код может стать громоздким и тяжело читаемым.
Для таких случаев есть конструкция <font color="#ff0000">Switch</font>. Она компактнее и понятнее:
```go
day := 3
switch day {
case 1:
    fmt.Println("Monday")
case 2:
    fmt.Println("Tuesday")
case 3:
    fmt.Println("Wednesday")
default:
    fmt.Println("Unknown day")
}
```
**Вывод:**
```Run
Wednesday
```

### **⚠️ В одном <font color="#d99694">case</font> может быть несколько значений:**
```go
day := 6
switch day {
case 6, 7:
    fmt.Println("Weekend")
default:
    fmt.Println("Weekday")
}
```

### <font color="#ff0000">Switch</font> без выражения
Обычно <font color="#ff0000">switch</font> сравнивает одно значение с набором вариантов.
Но в [[1.0. Go Basics|Go]] есть удобный режим:<mark style="background:rgba(240, 200, 0, 0.2)"> <font color="#ff0000">switch</font> без аргумента</mark>.

Тогда каждый <font color="#d99694">case</font> — это отдельное условие (почти как <font color="#ff0000">if</font>, только чище):
```go
age := 20
switch {
case age < 7:
    fmt.Println("Kindergarten")
case age < 18:
    fmt.Println("School")
case age < 23:
    fmt.Println("University")
default:
    fmt.Println("Work")
}
```
Вывод:
```Run
University
```

# Отличия <font color="#ff0000">switch</font> от <font color="#ff0000">if else</font>:

- Если условия зависят **от разных [[1.1. Переменные|переменных]]** или сложные (с логикой <font color="#ff0000">&&</font>, <font color="#ff0000">||</font>) → лучше использовать <font color="#ff0000">if else</font>.
- Если проверяем **одно и то же значение** (*например, день недели, месяц, команду в меню*) → удобнее использовать <font color="#ff0000">switch</font>, код будет короче и понятнее.

Проще говоря:
- <font color="#ff0000">if else</font> = универсальный инструмент «*на все случаи жизни*».
- <font color="#ff0000">switch</font> = быстрый и аккуратный способ, когда у тебя одно значение и несколько вариантов.

# Циклы ~ <font color="#ff0000">for</font> ~

В [[1.0. Go Basics|Go]] есть только **одна конструкция цикла — <font color="#ff0000">for</font>**.
Но она настолько универсальна, что может работать и как привычный <font color="#ff0000">for</font>, и как <font color="#ff0000">while</font>, и даже как бесконечный цикл.

Цикл нужен, когда мы хотим повторять какое-то действие несколько раз: *пройтись по числам, проверить список задач, нарисовать таблицу.*

### Цикл ~ <font color="#ff0000">for</font> ~ со счётчиком:
```go
for i := 1; i <= 5; i++ {
    fmt.Println(i)
}
```
- `i := 1` — это старт, начинаем с числа 1.
- `i <= 5` — условие, пока `i` меньше или равно 5, цикл будет работать.
- `i++` — это «шаг»; после каждой итерации мы увеличиваем `i` на единицу.

### Цикл ~ <font color="#ff0000">for</font> ~ с условием ( аналог <font color="#ff0000">while</font> ):
**⚠️ Используется когда начальное значение переменной известно заранее**
```go
i := 1
for i <= 5 {
    fmt.Println(i)
    i++
}
```
- Здесь `i` увеличивается внутри тела цикла.

### Бесконечный цикл ~ <font color="#ff0000">for</font> ~ :
```go
for {
    fmt.Println("Infinity loop")
}
```
- Такой цикл никогда не остановится сам по себе.
**⚠️ Обязательно должен быть способ выйти из бесконечного цикла: команда <font color="#ff0000">break</font> или завершение программы, иначе программа зависнет и будет бесконечно печатать.**

---
# Управление циклом: ~ <font color="#ff0000">break</font> ~ <font color="#ff0000">continue</font> ~

##### <font color="#ff0000">break</font> — полностью прерывает цикл.
##### <font color="#ff0000">continue</font> — пропускает текущую итерацию и переходит к следующей.

**Пример:**
```go
for i := 1; i <= 5; i++ {
    if i == 3 {
        continue // когда i = 3, пропустим печать
    }
    if i == 5 {
        break // на i = 5 цикл остановится
    }
    fmt.Println(i)
}
```
- `i = 1` → печатаем `1`.
- `i = 2` → печатаем `2`.
- `i = 3` → сработал `continue`, ничего не печатаем.
- `i = 4` → печатаем `4`.
- `i = 5` → сработал `break`, цикл завершается.

**Вывод:**
```Run
1
2
4
```

# Метки ~ <font color="#ff0000">Label</font> ~ в циклах, <font color="#ff0000">break</font> и <font color="#ff0000">continue</font>

<font color="#ff0000">Label</font> - способ "пометить" цикл, чтобы из вложенного цикла можно было вернуться к нему с помощью <font color="#ff0000">break</font> или <font color="#ff0000">continue</font>.

##### Пример цикла с <font color="#ff0000">Label</font> и <font color="#ff0000">continue</font>:
```Go
package main  
import "fmt"  
func main() {  
Label1:  
    for i := 1; i <= 20; i++ {  
       for j := 1; j <= 10; j++ {  
          fmt.Println("I: ", i, "J: ", j)  
          if i >= 10 {  
             continue Label1  
          }  
       }  
    }  
}
```
- `i` изменяется от **1** до **20**, `j` — от **1** до **10**.
- Пока `i < 10` → внутренний цикл выполняется полностью (`j = 1...10`).
- Если `i >= 10` → срабатывает `continue Label1`.
- `continue Label1` → сразу переходит к следующей итерации __внешнего__ цикла (`i`), пропуская оставшиеся итерации внутреннего (`j`).
- `Label1` → метка, указывающая, к какому именно циклу должен перейти `continue`.
---
# Вложенные циклы

**Один цикл можно поставить внутрь другого:**
```go
for i := 1; i <= 3; i++ {
    for j := 1; j <= 3; j++ {
        fmt.Print(i*j, " ")
    }
    fmt.Println()
}
```
- Когда `i = 1`, внутренний цикл перебирает `j = 1, 2, 3` → выводит `1 2 3`.
- Когда `i = 2` → выводит `2 4 6`.
- Когда `i = 3` → выводит `3 6 9`.
**Вывод:**
```Run
1 2 3
2 4 6
3 6 9
```

# Цикл ~ <font color="#ff0000">range</font> ~

Необходим, когда нужно пройтись по строке или по массиву:
```go
word := "Go"
for i, ch := range word {
    fmt.Println(i, string(ch))
}
```
- `i` — это индекс (позиция символа).
- `ch` — символ (тип `rune`).
- `i = 0`, `ch = 'G'` → выводим `0 G`.
- `i = 1`, `ch = 'o'` → выводим `1 o`.
**Вывод:**
```
0 G
1 o
```
**⚠️ <font color="#ff0000">range</font> возвращает символы в формате [[1.2. Типы данных#<font color=" ff0000">byte</font> и <font color=" ff0000">rune</font> в 1.0. Go Basics Go|rune]], поэтому мы преобразовали их в строку через [[1.2. Типы данных#3. Строки ~ <font color=" ff0000">string</font> ~|string(ch)]].**
