# Run
Окно и инструмент [[2.0.0. JetBrains Goland|GoLand]] для запуска приложения, тестов или отдельных файлов без использования терминала.
# Основные возможности
- Запуск Go-приложений.
- Запуск отдельных файлов и пакетов.
- Запуск тестов.
- Передача аргументов командной строки.
- Настройка переменных окружения.
- Просмотр вывода программы.
- Повторный запуск приложения.
# Run Configuration
Перед запуском [[2.0.0. JetBrains Goland|GoLand]] использует **Run Configuration** — набор настроек, определяющий, как именно будет запускаться программа.

**Run Configuration может содержать:**
- имя конфигурации;
- запускаемый файл или пакет;
- рабочую директорию (Working Directory);
- переменные окружения (Environment Variables);
- аргументы командной строки (Program Arguments);
- параметры Go Tool.

Для одного проекта можно создать несколько различных конфигураций запуска.
# Запуск программы
Запустить приложение можно несколькими способами:
- нажать зелёную кнопку ▶ Run;
- выбрать **Run → Run...**;
- щёлкнуть по зелёному треугольнику рядом с функцией `main()`.

# Окно Run

После запуска открывается окно **Run**, в котором отображаются:
- вывод программы (`stdout`);
- сообщения об ошибках (`stderr`);
- код завершения процесса (Exit Code);
- время выполнения.

Также доступны кнопки:
- **Restart** — перезапустить программу;
- **Stop** — остановить выполнение;
- **Rerun** — повторный запуск.

---
# GoLand позволяет запускать:
- Go Application;
- отдельный Go-файл;
- Go Package;
- Unit Tests;
- Benchmark Tests.

|  Запустить программу  | `⌃ R`  |

| Остановить выполнение | `⌘ F2` |

# Окно Run используется для:

- запуска приложений;
