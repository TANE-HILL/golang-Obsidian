# Горячие клавиши [[2.0.0. JetBrains Goland|IDE JetBrains Goland]]

### 1. [[2.1.0. Project|Project]] HOTKEYS

|     Действие      |    macOS    |
| :---------------: | :---------: |
|  Открыть Project  |  **`⌘ 1`**  |
|    Поиск файла    | **`⇧ ⌘ O`** |
| Search Everywhere |  **`⇧ ⇧`**  |
| Новая папка/файл  |  **`⌃ N`**  |
### 2. [[2.1.1. Editor|Editor]] HOTKEYS

|         Действие         |  macOS  |
| :----------------------: | :-----: |
|    Форматировать код     | `⌥ ⌘ L` |
|  Комментировать строку   |  `⌘ /`  |
|    Дублировать строку    |  `⌘ D`  |
|      Удалить строку      |  `⌘ ⌫`  |
| Переместить строку вверх | `⌥ ⇧ ↑` |
| Переместить строку вниз  | `⌥ ⇧ ↓` |
### 3. [[2.1.2. Terminal|Terminal]] HOTKEYS

|     Действие     |      macOS       |
| :--------------: | :--------------: |
| Открыть Terminal |     `⌥ F12`      |
|  Очистить экран  | `clear` или `⌘K` |
|  Новая вкладка   |      `⌘ T`       |
### 4. [[2.1.3. Run|Run]] HOTKEYS

|       Действие        | macOS  |
| :-------------------: | :----: |
|  Запустить программу  | `⌃ R`  |
|   Повторный запуск    | `⌃ R`  |
| Остановить выполнение | `⌘ F2` |
### 5. [[2.1.4. Debug|Debug]] HOTKEYS

|     Действие      |  macOS  |
| :---------------: | :-----: |
|       Debug       |  `⌃ D`  |
|     Continue      | `⌥ ⌘ R` |
|     Step Over     |  `F8`   |
|     Step Into     |  `F7`   |
|     Step Out      | `⇧ F8`  |
| Toggle Breakpoint | `⌘ F8`  |

|       Действие        |  macOS  |

|        Commit         |  `⌘ K`  |
