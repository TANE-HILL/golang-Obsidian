# Editor
Основная рабочая область [[2.0.0. JetBrains Goland|GoLand]], в которой происходит написание, редактирование и просмотр кода.

# Основные возможности
- Редактирование исходного кода.
- Подсветка синтаксиса.
- Автоматическое форматирование кода.
- Автодополнение (Code Completion).
- Быстрая навигация по коду.
- Отображение ошибок и предупреждений в реальном времени.
- Работа с несколькими вкладками.
- 
## Подсветка кода
Editor автоматически выделяет:
- ключевые слова языка;
- строки;
- числа;
- комментарии;
- функции;
- структуры;
- интерфейсы;
- пакеты;
- ошибки синтаксиса.

Это делает код более читаемым и помогает быстрее находить ошибки.
# Code Completion
Автодополнение предлагает:
- функции;
- методы;
- поля структур;
- интерфейсы;
- переменные;
- импортируемые пакеты.
---
# Инспекции кода

[[2.0.0. JetBrains Goland|GoLand]] анализирует код во время набора и может:

- обнаружить синтаксические ошибки;
- показать неиспользуемые переменные;
- предложить импорт недостающего пакета;
- предупредить о потенциальных ошибках.
---
# Навигация
Из Editor можно быстро перейти:
- к определению функции;
- к реализации интерфейса;
- к объявлению переменной;
- к использованию элемента.
---
# Работа со вкладками
Editor поддерживает несколько одновременно открытых файлов.

Можно:
- закреплять вкладки (Pin Tab);
- закрывать отдельные или все вкладки;
- разделять окно редактора по горизонтали и вертикали;
- перетаскивать вкладки между группами.
---
# Горячие клавиши окна Editor

|         Действие         |  macOS  |
| :----------------------: | :-----: |
|    Форматировать код     | `⌥ ⌘ L` |
|  Комментировать строку   |  `⌘ /`  |
|    Дублировать строку    |  `⌘ D`  |
|      Удалить строку      |  `⌘ ⌫`  |
| Переместить строку вверх | `⌥ ⇧ ↑` |
| Переместить строку вниз  | `⌥ ⇧ ↓` |
