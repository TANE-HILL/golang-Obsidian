# Terminal
Встроенный терминал [[2.0.0. JetBrains Goland|GoLand]], позволяющий выполнять команды операционной системы, не покидая IDE.

# Основные возможности
- Выполнение команд оболочки (Shell).
- Запуск Go-приложений.
- Работа с Git.
- Управление зависимостями проекта.
- Запуск утилит и скриптов.
- Работа с Docker, Kubernetes и другими CLI-инструментами.

# Что можно делать:

### 1. Запуск программы
```bash
go run .
```
или
```bash
go run main.go
```
### 2. Сборка проекта
```bash
go build
```
### 3. Запуск тестов
```bash
go test ./...
```
### 4. Управление зависимостями
```bash
go mod init
go mod tidy
go get
go mod download
```
### 5. Работа с Git
```bash
git status
git add .
git commit -m "Initial commit"
git push
```
---
# Преимущества встроенного Terminal
- Не нужно переключаться между IDE и отдельным терминалом.
- Терминал открывается сразу в директории текущего проекта.
- Поддерживает все команды, доступные в обычной консоли.
- Можно открыть несколько вкладок терминала.
---
# Горячие клавиши окна Terminal

|     Действие     |      macOS       |
| :--------------: | :--------------: |
| Открыть Terminal |     `⌥ F12`      |
|  Очистить экран  | `clear` или `⌘K` |
|  Новая вкладка   |      `⌘ T`       |
