# Project
Основное окно навигации по проекту [[2.0.0. JetBrains Goland|JetBrains GoLand]]. Отображает структуру каталогов и файлов, позволяя быстро перемещаться между ними.

# Основные возможности
- Просмотр структуры проекта.
- Создание файлов и папок.
- Создание Go-пакетов.
- Переименование и удаление файлов.
- Перемещение файлов и директорий.
- Быстрый поиск нужного файла.
- Просмотр Git-статуса файлов.
- Работа с несколькими модулями проекта.

# Контекстное меню (ПКМ)
Наиболее часто используемые действия:
- **New** — создание файла, папки, Go File, Go Package и др.
- **Refactor** — безопасное переименование и перемещение.
- **Copy / Paste** — копирование файлов.
- **Delete** — удаление.
- **Find Usages** — поиск использования файла или элемента.
- **Reveal in Finder / Explorer** — открыть папку в проводнике.
- **Git** — операции с системой контроля версий.

# Цветовые индикаторы Git
- **Зелёный** — новый файл.
- **Синий** — изменённый файл.
- **Красный** — файл помечен на удаление.
- **Серый** — файл игнорируется (`.gitignore`).

# Полезные функции
- **Auto Scroll from Source** — автоматически выделяет текущий файл в дереве проекта.
- **Auto Scroll to Source** — открывает файл при выборе его в дереве.
- Возможность закреплять избранные папки.
- Сворачивание и разворачивание каталогов.
