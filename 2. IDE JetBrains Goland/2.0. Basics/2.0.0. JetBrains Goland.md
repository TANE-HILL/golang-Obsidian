# GoLand
Cпециализированная IDE (интегрированная среда разработки) на Go от JetBrains.

# Основные возможности IDE
- **Подсветка синтаксиса**
	IDE выделяет ключевые слова, переменные, типы, делает код читаемым.
- **Автодополнение (IntelliSense)**
	IDE подсказывает функции, переменные и типы, чтобы не набирать всё вручную.
- **Быстрый переход к функциям и файлам**
	можно мгновенно перейти к нужному месту в коде.
- **Подсветка ошибок в реальном времени**
	если ты допустил ошибку в синтаксисе, IDE сразу подсветит её и подскажет, как исправить.
- **Интегрированный терминал**
	можно запускать программу прямо из IDE, не выходя в консоль.
