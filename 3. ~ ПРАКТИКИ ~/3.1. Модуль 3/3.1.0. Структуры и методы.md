# Задача 1

**Формулировка:**
Создайте несколько пользователей и выведите имена тех, кто старше 18 лет.

- Создать тип `User` с полями `Name` и `Age`.
- Добавить пользователей:
    - Vasya — 19
    - Masha — 17
    - Petr — 25
    - Olya — 30
- Сохранить их в слайс.
- Пройти циклом и вывести имена, где `Age > 18`.

**Решение:**
```Go
package main  
import "fmt"  
type users struct {  
    name string  
    age  int  
}  
func main() {  
    people := []users{}  
    people = append(people, users{"Vasya", 19}, users{"Masha", 17}, users{"Petr", 25}, users{"Olya", 30})  
    for _, person := range people {  
       if person.age >= 18 {  
          fmt.Println(person.name)  
       }  
    }  
}
```
**Вывод:**
```Run
Vasya
Petr
Olya
```

# Задача 2

**Формулировка:**
Добавьте нового пользователя в существующий слайс и выведите обновлённое количество пользователей.

- Создать слайс `users := []User{...}` из трёх пользователей.
- Добавить четвёртого при помощи `append`.
- Вывести количество элементов (`len(users)`).

**Решение:**
```Go
package main  
import "fmt"  
type users struct {  
    name string  
    age  int  
}  
func main() {  
    people := []users{}  
    people = append(people, users{"Vasya", 19}, users{"Masha", 17}, users{"Petr", 25}, users{"Olya", 30})  
    fmt.Printf("Количество пользователей: %v", len(people))  
}
```
**Вывод:**
```Run
Количество пользователей: 4
```
