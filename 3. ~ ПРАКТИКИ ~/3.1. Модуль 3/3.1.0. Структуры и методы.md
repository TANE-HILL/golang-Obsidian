# Задача 1

**Формулировка:**
Создайте несколько пользователей и выведите имена тех, кто старше 18 лет.

- Создать тип `User` с полями `Name` и `Age`.
- Добавить пользователей:
    - Vasya — 19
    - Masha — 17
    - Petr — 25
    - Olya — 30
- Сохранить их в слайс.
- Пройти циклом и вывести имена, где `Age > 18`.

**Решение:**
```Go
package main  
import "fmt"  
type users struct {  
    name string  
    age  int  
}  
func main() {  
    people := []users{}  
    people = append(people, users{"Vasya", 19}, users{"Masha", 17}, users{"Petr", 25}, users{"Olya", 30})  
    for _, person := range people {  
       if person.age >= 18 {  
          fmt.Println(person.name)  
       }  
    }  
}
```
**Вывод:**
```Run
Vasya
Petr
Olya
```

# Задача 3

**Формулировка:**
Напишите функцию, которая ищет пользователя по имени в слайсе.

```go
func FindUser(users []User, name string) *User
```

- Если пользователь найден — вернуть указатель на него.
- Если нет — вернуть `nil`.
- Проверить поиск по именам `"Petr"` и `"Anna"`.

**Решение:**
```Go
package main  
import "fmt"  
type User struct {  
    name string  
    age  int  
}  
func FindUser(users []User, name string) *User {  
    for i := range users {  
       if users[i].name == name {  
          return &users[i]  
       }  
    }  
    return nil  
}  
func checkName(user *User) {  
    if user == nil {  
       fmt.Printf("Не найден\n")  
    } else {  
       fmt.Printf("Найден: %v\n", user.name)  
    }  
}  
func main() {  
    users := []User{}  
    users = append(users, User{"Vasya", 19}, User{"Masha", 17}, User{"Petr", 25}, User{"Olya", 30})  
    fmt.Printf("Количество пользователей: %v\n", len(users))  
    a := FindUser(users, "Vasya")  
    b := FindUser(users, "Anna")  
    checkName(a)  
    checkName(b)  
}
```
**Вывод:**
```Run
Количество пользователей: 4
Найден: Vasya
Не найден
```

# Задача 7

**Формулировка:**
Создайте метод, возвращающий информацию о пользователе одной строкой.

```go
func (u User) Card() string
```

- Формат: `"Vasya (25) <v@mail.com>"`.
- Вывести результат в консоль.

**Решение:**
```Go
package main  
import (  
    "fmt"  
    "strconv")  
type User struct {  
    Name  string  
    Age   int  
    Email string  
}  
func (u User) Card() string {  
    result := u.Name + " (" + strconv.Itoa(u.Age) + ") " + u.Email  
    return result  
}  
func main() {  
    users := []User{  
       {Name: "Vasya", Age: 24, Email: "VasyaVpiska@mail.ru"},  
       {Name: "Masha", Age: 14, Email: "Mashka_Sha@gmail.com"},  
       {Name: "Anya", Age: 26, Email: "Anna_annA@rembler.ru"},  
       {Name: "Kolya", Age: 22, Email: "Nikolay0101@gmail.com"},  
    }  
    fmt.Println(users[1].Card())  
}
```
**Вывод:**
```Run
Masha (14) Mashka_Sha@gmail.com
```

# Задача 8

**Формулировка:**
Создайте слайс пользователей и увеличьте возраст всех на 1 с помощью метода `Birthday()`.

- Vasya — 25
- Masha — 17
- Petr — 30
- Пройти циклом по всем пользователям и вызвать метод `Birthday()`.

**Решение:**
```Go
package main  
import "fmt"  
type User struct {  
    Name  string  
    Age   int  
    Email string  
}  
func (u *User) Birthday() {  
    u.Age++  
}  
func main() {  
    users := []User{  
       {Name: "Vasya", Age: 24, Email: "VasyaVpiska@mail.ru"},  
       {Name: "Masha", Age: 14, Email: "Mashka_Sha@gmail.com"},  
       {Name: "Anya", Age: 26, Email: "Anna_annA@rembler.ru"},  
       {Name: "Kolya", Age: 22, Email: "Nikolay0101@gmail.com"},  
    }  
    for _, user := range users {  
       fmt.Printf("Возраст %v - %v\n", user.Name, user.Age)  
       user.Birthday()  
       fmt.Printf("Новый возраст %v - %v\n", user.Name, user.Age)  
    }  
}
```
**Вывод:**
```Run
Возраст Vasya - 24
Новый возраст Vasya - 25
Возраст Masha - 14
Новый возраст Masha - 15
Возраст Anya - 26
Новый возраст Anya - 27
Возраст Kolya - 22
Новый возраст Kolya - 23
```

# Задача 9

**Формулировка:**
Выведите карточки (`Card()`) только совершеннолетних пользователей.

- Vasya — 25
- Masha — 15
- Olya — 22
- Используйте `IsAdult()` и `Card()`.

**Решение:**
```Go
package main  
import (  
    "fmt"  
    "strconv")  
type User struct {  
    Name  string  
    Age   int  
    Email string  
}  
func (u User) IsAdult() bool {  
    if u.Age >= 18 {  
       return true  
    }  
    return false  
}  
func (u User) Card() string {  
    result := u.Name + " (" + strconv.Itoa(u.Age) + ") " + u.Email  
    return result  
}  
func main() {  
    users := []User{  
       {Name: "Vasya", Age: 24, Email: "VasyaVpiska@mail.ru"},  
       {Name: "Masha", Age: 14, Email: "Mashka_Sha@gmail.com"},  
       {Name: "Anya", Age: 26, Email: "Anna_annA@rembler.ru"},  
       {Name: "Kolya", Age: 18, Email: "Nikolay0101@gmail.com"},  
    }  
    for _, user := range users {  
       if user.IsAdult() {  
          a := user.Card()  
          fmt.Println(a)  
       }  
    }  
}
```
**Вывод:**
```Run
Vasya (24) VasyaVpiska@mail.ru
Anya (26) Anna_annA@rembler.ru
Kolya (18) Nikolay0101@gmail.com
```

# Задача 10

**Формулировка:**
Проверьте, что метод с указателем изменяет данные, а без указателя — нет.

```go
func (u User) WrongBirthday()
func (u *User) RealBirthday()
```

- Vasya — 29
- Вызвать оба метода и вывести возраст после каждого.

**Решение:**
```Go
package main  
import "fmt"  
type User struct {  
    Name  string  
    Age   int  
    Email string  
}  
func (u User) WrongBirthday() {  
    u.Age++  
}  
func (u *User) RealBirthday() {  
    u.Age++  
}  
func main() {  
    users := []User{  
       {Name: "Vasya", Age: 24, Email: "VasyaVpiska@mail.ru"},  
       {Name: "Masha", Age: 14, Email: "Mashka_Sha@gmail.com"},  
       {Name: "Anya", Age: 26, Email: "Anna_annA@rembler.ru"},  
       {Name: "Kolya", Age: 18, Email: "Nikolay0101@gmail.com"},  
    }  
    for _, user := range users {  
       fmt.Printf("%v до Дня Рождения: %v\n", user.Name, user.Age)  
       user.WrongBirthday()  
       fmt.Printf("%v после фейкового Дня Рождения: %v\n", user.Name, user.Age)  
       user.RealBirthday()  
       fmt.Printf("%v после настоящего Дня Рождения: %v\n", user.Name, user.Age)  
    }  
}
```
**Вывод:**
```Run
Vasya до Дня Рождения: 24
Vasya после фейкового Дня Рождения: 24
Vasya после настоящего Дня Рождения: 25
Masha до Дня Рождения: 14
Masha после фейкового Дня Рождения: 14
Masha после настоящего Дня Рождения: 15
Anya до Дня Рождения: 26
Anya после фейкового Дня Рождения: 26
Anya после настоящего Дня Рождения: 27
Kolya до Дня Рождения: 18
Kolya после фейкового Дня Рождения: 18
Kolya после настоящего Дня Рождения: 19
```

# Задача 11

**Формулировка:**
Добавьте в `User` поле `Address` и выведите город пользователя.

```go
type Address struct {
    City, Street string
}
```

- Olya — 25, Address: Moscow, Tverskaya.

- Вывести фразу:
    `Olya живёт в Moscow на улице Tverskaya`.

**Решение:**
```Go
package main  
import "fmt"  
type User struct {  
    Name string  
    Age  int  
    Addr Address  
}  
type Address struct {  
    City, Street string  
}  
func main() {  
    users := []User{  
       {"Sasha", 24, Address{"Chelyabinsk", "Kirova"}},  
       {"Olya", 14, Address{"Moscow", "Tverskaya"}},  
       {"Vadim", 28, Address{"Omsk", "Pushkina"}},  
    }  
    for _, user := range users {  
       fmt.Printf("%v живёт в %v на улице %v\n", user.Name, user.Addr.City, user.Addr.Street)  
    }  
}
```
**Вывод:**
```Run
Sasha живёт в Chelyabinsk на улице Kirova
Olya живёт в Moscow на улице Tverskaya
Vadim живёт в Omsk на улице Pushkina
```
