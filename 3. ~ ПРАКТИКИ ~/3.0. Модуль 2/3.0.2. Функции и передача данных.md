- Пройтись по всем элементам.
- Вернуть наименьшее значение.

```Go
package main

func minAll(nums ...int) (int, bool) {  
    if len(nums) == 0 {  
       return 0, false  
    }  
    min := nums[0]  
    for _, num := range nums[1:] {  
       if min > num {  
          min = num  
       }  
    }  
    return min, true  
}  
func main() {  
    min, ok := minAll()  
    if !ok {  
       fmt.Println("Список пуст")  
    } else {  
       fmt.Println(min)  
    }  
    min, ok = minAll(3, 7, 1, 8, 2)  
    if !ok {  
       fmt.Println("Список пуст")  
    } else {  
       fmt.Println(min)  
    }  
}
```
**Вывод:**
```Run
Список пуст
1
```

**Формулировка:**
Создайте функцию `report(tasks int) (int, int, int)`, распределяющую задачи по категориям:
- кратные 3 → doing
- чётные → done
- остальные → todo

- Пройти цикл от 1 до `tasks`.
- Подсчитать каждую категорию.
- Вернуть три значения.

**Решение:**
```Go
package main  
import "fmt"  
func report(tasks int) (int, int, int) {  
    todo, doing, done := 0, 0, 0  
    for tasks > 0 {  
       switch {  
       case tasks%3 == 0:  
          doing++  
       case tasks%2 == 0:  
          done++  
       default:  
          todo++  
       }  
       tasks--  
    }  
    return todo, doing, done  
}  
func main() {  
    t, d, f := report(15)  
    fmt.Println("Todo:", t, "Doing:", d, "Done:", f)  
}
```
**Вывод:**
```Run
Todo: 5 Doing: 5 Done: 5
```

# Задача 25

**Формулировка:**
Создайте функцию `validatePassword(password string) bool`, которая проверяет, является ли пароль **надёжным**.

Пароль считается надёжным, если в нём есть:
- хотя бы **одна цифра** (`0–9`);
- хотя бы **одна строчная буква** (`a–z`);
- хотя бы **одна заглавная буква** (`A–Z`);
- длина не меньше **8 символов**.

Функция должна вернуть `true`, если все условия выполнены, иначе — `false`.

- Перебрать символы строки `password`.
- Проверить, есть ли цифра, маленькая и большая буква.
- После цикла вернуть `true`, если все проверки прошли.

**Решение:**
```Go
package main  
import "fmt"  
func validatePassword(password string) bool {  
    num, big, small := 0, 0, 0  
    if len(password) < 8 {  
       return false  
    }  
    for _, r := range password {  
       switch {  
       case 'a' <= r && r <= 'z':  
          small++  
       case 'A' <= r && r <= 'Z':  
          big++  
       case '0' <= r && r <= '9':  
          num++  
       }  
    }  
    switch {  
    case num == 0 || small == 0 || big == 0:  
       return false  
    default:  
       return true  
    }  
}  
func main() {  
    fmt.Println(validatePassword("GoLang123"))  // true  
    fmt.Println(validatePassword("golang"))     // false  
    fmt.Println(validatePassword("GOLANG2025")) // false  
}
```
**Вывод:**
```Run
true
false
false
```

# Задача 27

**Формулировка:**
Создайте функцию `isPalindrome(n int) bool`, проверяющую, является ли число палиндромом.

- Сохранить исходное значение.
- Перевернуть число: в цикле брать последнюю цифру и собирать новое.
- Сравнить с оригиналом.

**Решение:**
```Go
package main  
import "fmt"  
func isPalindrome(n int) bool {  
    dir, rev := n, 0  
    for n > 0 {  
       rev = rev*10 + n%10  
       n /= 10  
    }  
    return dir == rev  
}  
func main() {  
    fmt.Println(isPalindrome(121))  
    fmt.Println(isPalindrome(123))  
}
```
**Вывод:**
```Run
true
false
```
