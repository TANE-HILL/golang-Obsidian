# 1. Анализатор текста ⭐⭐⭐

Напиши функцию
```
analyze(text string)
```

Она должна **ничего не возвращать**, а вывести:
```
Строчных букв: 12
Заглавных букв: 3
Цифр: 5
Пробелов: 4
Других символов: 2
```

Нельзя использовать пакет `unicode`.

---
**Решение:**

```Go
package main  
import "fmt"  
func analyze(text string) (int, int, int, int, int) {  
    lower, upper, num, space, other := 0, 0, 0, 0, 0  
    for _, r := range text {  
       switch {  
       case (r >= 'a' && r <= 'z') || (r >= 'а' && r <= 'я') || r == 'ё':  
          lower++  
       case (r >= 'A' && r <= 'Z') || (r >= 'А' && r <= 'Я') || r == 'Ё':  
          upper++  
       case r >= '0' && r <= '9':  
          num++  
       case r == ' ':  
          space++  
       default:  
          other++  
       }  
    }  
    return lower, upper, num, space, other  
}  
func main() {  
    fmt.Println(analyze("Работа над Go началась в 2007 в компании Google. Роберт Гриземер (Robert Griesemer), Роб Пайк (Rob Pike) и Кен Томпсон (Ken Thompson)."))  
}
```

**Вывод:**
```Run
85 15 4 21 9
```

# 2. Мини-компрессор ⭐⭐⭐⭐

Напиши функцию

```
compress(text string) string
```

Она должна заменить подряд идущие одинаковые символы на символ и количество.

Например

```
aaabbccccdd

↓

a3b2c4d2
```

Еще пример

```
Helloooo!!!!

↓

Hel2o4!4
```

Нельзя использовать `strings.Builder` и `strings.Repeat`.

---

**Решение:**

```Go
package main
import "fmt"
func compress(text string) string {
    count, result := 1, ""
    for v := 0; v < len(text)-1; v++ {
        switch {
        case text[v] == text[v+1]:
            count++
            if v == len(text)-2 {
                result += string(text[v]) + string(byte(count+'0'))
            }
        default:
            result += string(text[v]) + string(byte(count+'0'))
            count = 1            
        }
    }
    return result
}
func main() {
    text := "aaabbccccdd"
    fmt.Println (compress(text))
}
```

**Вывод:**
```Run
a3b2c4d2
```

# 3. Оценка сложности парол] ⭐⭐⭐⭐

Напиши функцию

```
textScore(text string) int
```

Начальный счет — 0.

Добавить:

- +1 если есть цифры
- +1 если есть строчные буквы
- +1 если есть заглавные
- +1 если есть спецсимволы (`!@#$%^&*` и т.п.)
- +1 если длина больше 15 символов
- +1 если нет пробелов

Вернуть итоговый балл.

---

**Решение:**

```Go
package main
import "fmt"
func pswdScore(text string) {
    if len(text) < 8 {
        fmt.Printf("Длина пароля менее 8 символов недопустима!\nТекущая длина:%v\n", len(text))
        return
    }
    var num, low, up, symb, iNc bool
    inc := ""
    for _, r := range text {
        switch {
        case r == ' ':
            fmt.Printf("Пароль не может содержать пробелы.\n")
            return
        case r >= '0' && r <= '9':
            num = true
        case (r >= 'a' && r <= 'z'):
            low = true
        case (r >= 'A' && r <= 'Z'):
            up = true
        case r == '!' || r == '@' || r == '#' || r == '$' || r == '%' || r == '^' || r == '&' || r == '*':
            symb = true
        default:
            inc += string(r)
            iNc = true
        }
    }
    if !iNc && num && low && up && symb {
        fmt.Printf("Ваш пароль '%v' подходит!", text)
        return
    }
    if iNc {
        fmt.Printf("Пароль содержит недопустимые символы: '%v'\n", inc)
        }
    if !num {
        fmt.Printf("Пароль не содержит ни одной цифры.\n")
        }
    if !low {
        fmt.Printf("Пароль не содержит ни одной строчной буквы.\n")
        }
    if !up {
        fmt.Printf("Пароль не содержит ни одной заглавной буквы.\n")
        }
    if !symb {
        fmt.Printf("Пароль не содержит ни одного спецсимвола.\n")
        }
}
func main() {
    text := "sex!2ESe111"
    pswdScore (text)
}
```

**Вывод:**
```Run
Ваш пароль 'sex!2ESe111' подходит!
```

# 4. "Почти палиндром" ⭐⭐⭐⭐⭐

Напиши функцию

```
almostPalindrome(text string) bool
```

Вернуть `true`, если строка станет палиндромом после удаления **не более одного** символа.

Например

```
abca

↓

true
```

потому что можно удалить `c`.

А

```
abcdef
```

↓

```
false
```

---

**Решение:**

```Go
package main
import "fmt"
func ifPalindrome (text string) bool {
    for n := 0; n < len(text)/2; n++ {
        if text[n] != text[len(text)-1-n] {
            break
        }
        if n == len(text)/2-1 {
            return true
            }
        }
    return false
}
func almostPalindrome(text string) bool {
    for i, _ := range text {
        check := string(text[:i]) + string(text[i+1:])
        res := ifPalindrome(check)
        if res {
            return true
        }
    }
    return false
}
func main() {
    text := "isssdi"
    okay := almostPalindrome(text)
    fmt.Println(okay)
}
```

**Вывод:**
```Run
true
```

# 5. Умная статистика ⭐⭐⭐⭐

Напиши функцию

```
stats(nums ...int)
```

Она должна вывести

```
Минимум: 2
Максимум: 17
Количество четных: 6
Количество нечетных: 4
Среднее: 8.4
Есть ли отрицательные: Да
```

Все вычисляется за **один** проход по слайсу.

---

**Решение:**

```Go
package main
import "fmt"
func stats(nums []int) {
    var even, odd int
    average, ng := 0., false
    min, max := nums[0], nums[0]
    for _, v := range nums {
        if v != 0 {
            if v%2 == 0 {
                even++
            } else {
                odd++
            }
        }
        if v > max {
            max = v
        }
        if v < min {
            min = v
        }
        if v < 0 {
            ng = true
        }
        average += float64(v)
    }
    average /= float64(len(nums))
    fmt.Printf("Минимум: %v\nМаксимум: %v\nКоличество четных: %v\nКоличество нечетных: %v\nСреднее: %.2f\nЕсть ли отрицательные: %v", min, max, even, odd, average, ng)
}
func main() {
    s := []int {4, 6, 9, 12, -23, 2, 65, 9, 0, 3, 45, 4, 2}
    stats(s)
}
```

**Вывод:**
```Run
Минимум: -23
Максимум: 65
Количество четных: 6
Количество нечетных: 6
Среднее: 10.62
Есть ли отрицательные: true
```
