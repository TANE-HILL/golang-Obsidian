# Задача 1

**Формулировка:**
Создать массив из трёх чисел и вывести его длину и элементы.

- Объявить массив `[3]int`.
- Заполнить числами `10, 20, 30`.
- Вывести длину и содержимое массива.

**Решение:**
```Go
package main  
import "fmt"  
func main() {  
    a := [3]int{10, 20, 30}  
    fmt.Printf("Длина: %v\n%v", len(a), a)  
}
```
**Вывод:**
```Run
Длина: 3
[10 20 30]
```

# Задача 3

**Формулировка:**
Создать массив и вывести сумму первого и последнего элемента.

- Создать массив из длиной в 5 элементов. Заполнить его через range числами от 5 до 25 с шагом в 5 (5 10 15 20 25).
- Найти сумму `arr[0] + arr[len(arr)-1]`.
- Вывести результат.

**Решение:**
```Go
package main  
import "fmt"  
func main() {  
    a := [5]int{}  
    for i, v := range a {  
       a[i] = v + 5*(i+1)  
    }  
    fmt.Println(a[0] + a[len(a)-1])  
}
```
**Вывод:**
```Run
30
```

### Задача 5. Подсчёт нулей

**Формулировка:**
Подсчитать количество нулей в массиве.

- Пройти циклом по элементам.


```Run
Нулей: 4
```

# Задача 8.1.

**Формулировка:**
Создать слайс с помощью `make` и заполнить числами от 1 до 5.

- Создать `nums := make([]int, 5)`.
- Заполнить через цикл `range`, чтобы каждый элемент равнялся `i + 1`.
- Вывести слайс, длину и ёмкость.

**Решение:**
```Go
package main  
import "fmt"  
func main() {  
    a := make([]int, 5)  
    for i, _ := range a {  
       a[i] = i + 1  
    }  
    fmt.Printf("%v\nДлина: %v\nЁмкость: %v", a, len(a), cap(a))  
}
```
**Вывод:**
```Run
[1 2 3 4 5]
Длина: 5
Ёмкость: 5
```

# Задача 9 ⚠️

**Формулировка:**
Создать независимую копию массива через `copy`.

- Создать массив `[4]int{10,20,30,40}`.
- Скопировать его элементы в новый слайс.
- Изменить первый элемент нового слайса.
- Вывести оба массива, чтобы показать независимость.

**Решение:**
```Go
package main  
import "fmt"  
func main() {  
    a := [4]int{10, 20, 30, 40}  
    b := make([]int, len(a))  
    copy(b, a[:])  
    b[0] = 99  
    fmt.Printf("Исходный: %v\nКопия: %v", a, b)  
}
```
**Вывод:**
```Run
Исходный: [10 20 30 40]
Копия: [99 20 30 40]
```

# Задача 13

**Формулировка:**
Найти минимальное и максимальное значение в массиве.

- Создать `[6]int{15,2,30,7,9,100}`.
- Найти минимальное и максимальное значения.
- Вывести результат.

**Решение:**
```Go
package main  
import "fmt"  
func main() {  
    a := [6]int{15, 2, 30, 7, 9, 100}  
    min, max := a[0], a[0]  
    for _, v := range a {  
       switch {  
       case v < min:  
          min = v  
       case v > max:  
          max = v  
       }  
    }  
    fmt.Printf("Min: %d\nMax: %d", min, max)  
}
```
**Вывод:**
```Run
Min: 2
Max: 100
```
