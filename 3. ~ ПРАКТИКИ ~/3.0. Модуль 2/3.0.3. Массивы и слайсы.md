# Задача 1

**Формулировка:**
Создать массив из трёх чисел и вывести его длину и элементы.

- Объявить массив `[3]int`.
- Заполнить числами `10, 20, 30`.
- Вывести длину и содержимое массива.

**Решение:**
```Go
package main  
import "fmt"  
func main() {  
    a := [3]int{10, 20, 30}  
    fmt.Printf("Длина: %v\n%v", len(a), a)  
}
```
**Вывод:**
```Run
Длина: 3
[10 20 30]
```

# Задача 3

**Формулировка:**
Создать массив и вывести сумму первого и последнего элемента.

- Создать массив из длиной в 5 элементов. Заполнить его через range числами от 5 до 25 с шагом в 5 (5 10 15 20 25).
- Найти сумму `arr[0] + arr[len(arr)-1]`.
- Вывести результат.

**Решение:**
```Go
package main  
import "fmt"  
func main() {  
    a := [5]int{}  
    for i, v := range a {  
       a[i] = v + 5*(i+1)  
    }  
    fmt.Println(a[0] + a[len(a)-1])  
}
```
**Вывод:**
```Run
30
```

# Задача 4

**Формулировка:**
Найти минимальное значение в массиве.

- Создать `[6]int{8,2,7,4,9,1}`.
- Найти минимальное число и вывести его.

**Решение:**
```Go
package main  
import "fmt"  
func main() {  
    a := [6]int{8, 2, 7, 4, 9, 1}  
    min := a[0]  
    for _, v := range a {  
       if min > v {  
          min = v  
       }  
    }  
    fmt.Printf("Минимум: %v", min)  
}
```
**Вывод:**
```Run
Минимум: 1
```

# Задача 5

### Задача 5. Подсчёт нулей

**Формулировка:**
Подсчитать количество нулей в массиве.

- Создать `[8]int{1,0,5,0,7,0,9,0}`.
- Пройти циклом по элементам.
- Посчитать количество нулей и вывести.

**Решение:**
```Go
package main  
import "fmt"  
func main() {  
    a, z := [8]int{1, 0, 5, 0, 7, 0, 9, 0}, 0  
    for _, v := range a {  
       if v == 0 {  
          z++  
       }  
    }  
    fmt.Printf("Нулей: %v", z)  
}
```
**Вывод:**
```Run
Нулей: 4
```

# Задача 7

**Формулировка:**
Посчитать длину и ёмкость среза.

- Создать массив `[8]int{10,20,30,40,50,60,70,80}`.
- Сделать срез `arr[2:6]`.

package main

# Задача 8.1.

**Формулировка:**
Создать слайс с помощью `make` и заполнить числами от 1 до 5.

- Создать `nums := make([]int, 5)`.
- Заполнить через цикл `range`, чтобы каждый элемент равнялся `i + 1`.
- Вывести слайс, длину и ёмкость.

**Решение:**
```Go
package main  
import "fmt"  
func main() {  
    a := make([]int, 5)  
    for i, _ := range a {  
       a[i] = i + 1  
    }  
    fmt.Printf("%v\nДлина: %v\nЁмкость: %v", a, len(a), cap(a))  
}
```
**Вывод:**
```Run
[1 2 3 4 5]
Длина: 5
Ёмкость: 5
```

# Задача 8.2.

**Формулировка:**
Проверьте, как изменение элемента среза влияет на исходный массив.

- Создать массив `[5]int{100,200,300,400,500}`.
- Сделать срез `200,300`.
- Изменить первый элемент среза на `777`.
- Вывести массив и срез.

**Решение:**
```Go
package main
import "fmt"
func main() {
    a := [5]int{100, 200, 300, 400, 500}
    b := a[1:3]
    b[0] = 777
    fmt.Printf("Массив: %v\nСрез: %v", a, b)
}
```
**Вывод:**
```Run
Массив: [100 777 300 400 500]
Срез: [777 300]
```
