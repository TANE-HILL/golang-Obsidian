# Задача 1

**Формулировка:**
Создайте переменные `name` и `city`, затем выведите строку приветствия с их значениями.

- создать две переменные со строковыми значениями;
- объединить их через `+` или `fmt.Printf`;
- вывести строку `"Hello, Alex from Moscow!"`.

**Решение:**
```Go
package main
import "fmt"
func main() {
    name, city := "Alex", "Moscow"
    fmt.Printf("Hello, %s from %s!\n", name, city)
}
```
**Вывод:**
```Run
Hello, Alex from Moscow!
```

# Задача 15

**Формулировка:**
Создайте переменную `n := 12345` и найдите первую и последнюю цифру.

- вычислить `last := n % 10`;
- вычислить `first` делением на 10 до однозначного числа;
- вывести обе цифры.

**Решение:**
```Go
package main  
import "fmt"  
func main() {  
    n := 12345  
    last := n % 10  
    first := n  
    for first >= 10 {  
       first /= 10  
    }  
    fmt.Printf("First: %d\nLast: %d\n", first, last)  
}
```
**Вывод:**
```Run
First: 1
Last: 5
```
