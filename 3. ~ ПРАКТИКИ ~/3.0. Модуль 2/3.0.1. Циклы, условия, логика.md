# Задача 1

**Формулировка:**

```Go
    }  
}
```
**Вывод:**
```Run

```

# Задача 2

**Формулировка:**
Определите, холодно ли на улице.

- объявить `temp := 12`;
- если ниже 0 градусов → `Мороз`;
    если от 0 до 20 включительно → `Прохладно`;
    если выше 20 градусов → `Тепло`.

**Решение:**
```Go
package main  
import "fmt"  
func main() {  
    temp := 12  
    switch {  
    case temp < 0:  
       fmt.Printf("Мороз")  
    case temp <= 20:  
       fmt.Printf("Прохладно")  
    case temp > 20:  
       fmt.Printf("Тепло")  
    }  
}
```
**Вывод:**
```Run
Прохладно
```

```Go
             }  
          }  
```
```Go
       }  
    }  
```
```Go
}
```

# Задача 26

**Формулировка:**
Распределите задачи по трём категориям: `todo`, `doing`, `done`.

- `todo := 0`, `doing := 0`, `done := 0`;
- пройтись по числам 1..15:
    - если кратно 3 → `doing++`;
    - иначе если чётное → `done++`;
    - иначе → `todo++`;
- вывести статистику.

**Решение:**
```Go
package main  
import "fmt"  
func main() {  
    todo, doing, done := 0, 0, 0  
    for i := 1; i <= 15; i++ {  
       switch {  
       case i%3 == 0:  
          doing++  
       case i%2 == 0:  
          done++  
       default:  
          todo++  
       }  
    }  
    fmt.Printf("К выполнению: %d\nВ работе: %d\nГотово: %d", todo, doing, done)  
}
```
**Вывод:**
```Run
К выполнению: 5
В работе: 5
Готово: 5
```
