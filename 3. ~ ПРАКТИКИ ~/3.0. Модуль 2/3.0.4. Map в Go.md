# Задача 1

**Формулировка:**
Создать телефонную книгу, добавить несколько контактов и вывести общее количество записей.

- Инициализировать пустую карту пользователей через `make`.
- Добавить три контакта (имя → номер телефона).
- Вывести количество записей и весь список контактов.

**Решение:**
```Go
package main  
import "fmt"  
func main() {  
    nums := make(map[string]int)  
    nums["Masha"], nums["Petya"], nums["Vasya"] = 333, 222, 111  
    fmt.Printf("Количество контактов: %v\n%v", len(nums), nums)  
}
```
**Вывод:**
```Run
Количество контактов: 3
map[Masha:333 Petya:222 Vasya:111]
```

# Задача 4

**Формулировка:**
Создать список товаров и цен. Проверить наличие товар `"apple"` и товар `"lemon"` . Если он есть, вывести цену, иначе сообщение «Нет в наличии».

Создать ассортимент товаров и реализовать функцию, которая принимает название фрукта и возвращает сообщение о его наличии и флаг true/false.

Функция должна возвращать строку с сообщением и логическое значение, показывающее, найден ли товар.

- Инициализировать ассортимент: apple — 120, banana — 90, orange — 150, watermelon - 380, melon - 250.

- Написать функцию:
```go
func checkProduct(name string, goods map[string]int) (string, bool)
```
	которая возвращает сообщение вида `"Цена товара "яблоко": 120"` и `true`, если товар есть, либо `"Нет в наличии"` и `false`, если его нет.
    
- Вызвать функцию для `"apple"` и вывести результат в консоль. После проверки вывести функцию для  `"lemon"`

**Решение:**
```Go
package main  
import "fmt"  
func checkProduct(name string, goods map[string]int) (string, bool) {  
    if _, ok := goods[name]; ok {  
       return name, true  
    } else {  
       return "Нет в наличии.", false  
    }  
}  
func main() {  
    products := map[string]int{  
       "apple":      120,  
       "banana":     90,  
       "orange":     150,  
       "watermelon": 380,  
       "melon":      250,  
    }  
    a, b := checkProduct("apple", products)  
    if b == true {  
       fmt.Printf("Цена товара %v: %v", a, products[a])  
    } else {  
       fmt.Println(a)  
    }  
}
```
**Вывод:**
```Run
Цена товара apple: 120
```

# Задача 5

**Формулировка:**
Создать таблицу сотрудников и их зарплат. Повысить зарплату тем, у кого она меньше 1000, на 10%.

Реализовать логику в отдельной функции, которая принимает таблицу зарплат и возвращает обновлённые значения.

- Инициализировать данные о зарплатах: Vasya — 900, Petya — 1200, Masha — 800.
- Написать функцию, которая увеличивает зарплату всем, у кого значение < 1000, на 10%.

```go
func raiseSalaries(staff map[string]int)
```

- Вызвать функцию и вывести обновлённую таблицу зарплат.

**Решение:**
```Go
package main  
import "fmt"  
func raiseSalaries(staff map[string]int) {  
    for key, value := range staff {  
       if value < 1000 {  
          staff[key] += value / 10  
       }  
    }  
    return  
}  
func main() {  
    staff := map[string]int{  
       "Vasya": 900, "Petya": 1200, "Masha": 800,  
    }  
    raiseSalaries(staff)  
    for key, value := range staff {  
       fmt.Printf("%v - %v\n", key, value)  
    }  
}
```
**Вывод:**
```Run
Masha - 880
Vasya - 990
Petya - 1200
```

**Формулировка:**
Создать таблицу сотрудников и их зарплат.

Если сотрудник `"Vasya"` есть в таблице, увеличить его оклад на 15% и вывести обновлённую сумму.

Если такого сотрудника нет, вывести сообщение `"Нет сотрудника Vasya"`.

- Инициализировать данные: Vasya — 1200, Masha — 1500, Petya — 1000.
- Написать функцию, которая увеличивает зарплату выбранного сотрудника.

```go
func updateSalary(staff map[string]int, name string)
```

- Вызвать функцию для `"Vasya"` и вывести результат.

**Решение:**
```Go
package main  
import "fmt"  
func updateSalary(staff map[string]int, name string) {  
    if _, ok := staff[name]; ok {  
       staff[name] += staff[name] / 100 * 15  
       fmt.Printf("Зарплата %v увеличена: %v", name, staff[name])

       fmt.Printf("Сотрудника %v нет.", name)  
    }  
}  
func main() {  
    staff := map[string]int{  
       "Vasya": 1200, "Masha": 1500, "Petya": 1000,  
    }  
    updateSalary(staff, "Vasya")  
}
```
**Вывод:**
```Run
Зарплата Vasya увеличена: 1380
```

# Задача 8

**Формулировка:**
Создать ассортимент товаров и написать функцию, которая возвращает сумму всех цен.

- Инициализировать ассортимент:
    apple — 120, pear — 140, banana — 150, orange — 230.
- Реализовать функцию, которая считает общую сумму всех цен.

```go
func sumPrices(goods map[string]int) int
```

- Вызвать функцию и вывести результат

**Решение:**
```Go
package main  
import "fmt"  
func sumPrices(goods map[string]int) int {  
    sum := 0  
    for _, count := range goods {  
       sum += count  
    }  
    return sum  
}  
func main() {  
    goods := map[string]int{  
       "apple": 120, "pear": 140, "banana": 150, "orange": 230,  
    }  
    fmt.Printf("Общая сумма: %v\n", sumPrices(goods))  
}
```
**Вывод:**
```Run
Общая сумма: 640
```

# Задача 9

**Формулировка:**
Найти товар с максимальной ценой.

Реализовать отдельную функцию, возвращающую название и цену самого дорогого товара.

- Инициализировать ассортимент:
`apple — 120, banana — 90, orange — 150, pear — 140.`
- Написать функцию, которая возвращает название и цену самого дорогого товара.

```go
func maxItem(goods map[string]int) (string, int)
```

- Вызвать функцию и вывести результат в формате:

**Решение:**
```Go
package main  
import "fmt"  
func maxItem(goods map[string]int) (string, int) {  
    maxK, maxV := "", 0  
    for k, v := range goods {  
       if v > maxV {  
          maxK, maxV = k, v  
       }  
    }  
    return maxK, maxV  
}  
func main() {  
    goods := map[string]int{  
       "apple": 120, "pear": 140, "banana": 90, "orange": 150,  
    }  
    maxK, maxV := maxItem(goods)  
    fmt.Printf("Самый дорогой товар - %v: %v\n", maxK, maxV)  
}
```
**Вывод:**
```Run
Самый дорогой товар - orange: 150
```

# Задача 10

**Формулировка:**
Вывести товары, цена которых выше определённого порога.

Реализовать отдельную функцию, которая возвращает список названий таких товаров.

- Инициализировать ассортимент:
`apple — 120, banana — 90, orange — 150, pear — 140.`

- Написать функцию, которая возвращает слайс товаров, цена которых превышает указанный порог.

```go
func filterAbove(goods map[string]int, limit int) []string
```

- Вызвать функцию с порогом `100` и вывести все подходящие товары построчно.

**Решение:**
```Go
package main  
import "fmt"  
func filterAbove(goods map[string]int, limit int) []string {  
    result := make([]string, 0)  
    for i, v := range goods {  
       if v >= limit {  
          result = append(result, i)  
       }  
    }  
    for _, item := range result {  
       fmt.Println(item)  
    }  
    return result  
}  
func main() {  
    goods := map[string]int{  
       "apple": 120, "pear": 140, "banana": 90, "orange": 150,  
    }  
    filterAbove(goods, 100)  
}
```
**Вывод:**
```Run
orange
apple
pear
```
