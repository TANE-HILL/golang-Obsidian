# Задача 4

**Формулировка:**
Создать список товаров и цен. Проверить наличие товар `"apple"` и товар `"lemon"` . Если он есть, вывести цену, иначе сообщение «Нет в наличии».

Создать ассортимент товаров и реализовать функцию, которая принимает название фрукта и возвращает сообщение о его наличии и флаг true/false.

Функция должна возвращать строку с сообщением и логическое значение, показывающее, найден ли товар.

- Инициализировать ассортимент: apple — 120, banana — 90, orange — 150, watermelon - 380, melon - 250.

- Написать функцию:
```go
func checkProduct(name string, goods map[string]int) (string, bool)
```
	которая возвращает сообщение вида `"Цена товара "яблоко": 120"` и `true`, если товар есть, либо `"Нет в наличии"` и `false`, если его нет.
    
- Вызвать функцию для `"apple"` и вывести результат в консоль. После проверки вывести функцию для  `"lemon"`

**Решение:**
```Go
package main  
import "fmt"  
func checkProduct(name string, goods map[string]int) (string, bool) {  
    if _, ok := goods[name]; ok {  
       return name, true  
    } else {  
       return "Нет в наличии.", false  
    }  
}  
func main() {  
    products := map[string]int{  
       "apple":      120,  
       "banana":     90,  
       "orange":     150,  
       "watermelon": 380,  
       "melon":      250,  
    }  
    a, b := checkProduct("apple", products)  
    if b == true {  
       fmt.Printf("Цена товара %v: %v", a, products[a])  
    } else {  
       fmt.Println(a)  
    }  
}
```
**Вывод:**
```Run
Цена товара apple: 120
```

```Go
func avgGrades(students map[string]map[string]int) map[string]float64 {

       n := 0.
```

**Формулировка:**
Вывести три слова, которые встречаются чаще всего в тексте.

- Использовать частотную карту из предыдущей задачи.
- Написать функцию, которая возвращает `n` самых частых слов.

```go
func topWords(freq map[string]int, n int) []string
```

- Вывести результат в порядке убывания частоты.

**Решение:**
```Go
package main  
import (  
    "fmt"  
    "strings")  
func wordFreq(text string) map[string]int {  
    words := strings.FieldsFunc(text, func(r rune) bool {  
       return r == ' '  
    })  
    result := make(map[string]int)  
    for _, word := range words {  
       result[word]++  
    }  
    return result  
}  
func topWords(freq map[string]int, n int) []string {  
    used, result := map[string]bool{}, []string{}  
    for n > 0 {  
       maxCount, maxWord := 0, ""  
       for word, count := range freq {  
          if count >= maxCount && !used[word] {  
             maxCount = count  
             maxWord = word  
          }  
       }  
       used[maxWord] = true  
       result = append(result, maxWord)  
       n--  
    }  
    return result  
}  
func main() {  
    text := "go go is fun go is great"  
    result := wordFreq(text)  
    topWr := topWords(result, 3)  
    for _, v := range topWr {  
       fmt.Printf("%v - %v\n", v, result[v])  
    }  
}
```
**Вывод:**
```Run
go - 3
is - 2
great - 1
```

```Go
package main  
import "fmt"  
func copyWithout(m map[string]int, drop string) map[string]int {  
    for k, _ := range m {  
       if k == drop {  
          delete(m, k)  
       }  
    }  
    return m  
}  
func main() {  
    m := map[string]int{  
       "A": 1, "B": 2, "C": 3,  
    }  
    m2 := copyWithout(m, "B")
```

```Go
func unique(nums []int) []int {  
    result := []int{nums[0]}  
    for _, num := range nums[1:] {  
       pr := false  
       for _, v := range result {  
          if v == num {  
             pr = true  
          }  
       }  
       if pr == false {  
          result = append(result, num)  
       }  
  
    }  
    return result  
}
```


- Вывести результат построчно.

**Решение:**
```Go
package main  
import "fmt"  
func mergeAverage(m1, m2 map[string]float64) map[string]float64 {  
    m3 := map[string]float64{}  
    for k, v := range m1 {  
       m3[k] = v  
    }  
    for k, v := range m2 {  
       if _, ok := m3[k]; ok {  
          m3[k] = (m3[k] + v) / 2  
       } else {  
          m3[k] = v  
       }  
    }  
    return m3  
}

    m1 := map[string]float64{  
       "Vasya": 4.0, "Masha": 5.0,  
    }
```

```Go
func salesReport(sales [][]string) {

    for _, v := range sales {

            result[v[0]] = make([]int, 3)
        }
        result[v[0]][0]++
        for _, r := range v[2] {
            n = n*10 + int(r-'0')
        }
        result[v[0]][1] += n
```
