# 1. Анализатор текста ⭐⭐⭐

Напиши функцию
```
analyze(text string)
```

Она должна **ничего не возвращать**, а вывести:
```
Строчных букв: 12
Заглавных букв: 3
Цифр: 5
Пробелов: 4
Других символов: 2
```

Нельзя использовать пакет `unicode`.

# 2. Мини-компрессор ⭐⭐⭐⭐

Напиши функцию

```
compress(text string) string
```

Она должна заменить подряд идущие одинаковые символы на символ и количество.

Например

```
aaabbccccdd

↓

a3b2c4d2
```

Еще пример

```
Helloooo!!!!

↓

Hel2o4!4
```

Нельзя использовать `strings.Builder` и `strings.Repeat`.

---

# 3. Оценка сложности текста ⭐⭐⭐⭐

Напиши функцию

```
textScore(text string) int
```

Начальный счет — 0.

Добавить:

- +1 если есть цифры
- +1 если есть строчные буквы
- +1 если есть заглавные
- +1 если есть спецсимволы (`!@#$%^&*` и т.п.)
- +1 если длина больше 15 символов
- +1 если нет пробелов

Вернуть итоговый балл.

# 4. "Почти палиндром" ⭐⭐⭐⭐⭐

Напиши функцию

```
almostPalindrome(text string) bool
```

Вернуть `true`, если строка станет палиндромом после удаления **не более одного** символа.

Например

```
abca

↓

true
```

потому что можно удалить `c`.

А

```
abcdef
```

↓

```
false
```

# 5. Умная статистика ⭐⭐⭐⭐

Напиши функцию

```
stats(nums ...int)
```

Она должна вывести

```
Минимум: 2
Максимум: 17
Количество четных: 6
Количество нечетных: 4
Среднее: 8.4
Есть ли отрицательные: Да
```

Все вычисляется за **один** проход по слайсу.

# 6. Проверка скобок ⭐⭐⭐⭐⭐

Напиши функцию

```
validBrackets(text string) bool
```

Пока поддерживаются только

```
(
)
```

Примеры

```
(()())

↓

true
```

```
())(

↓

false
```

```
((())

↓

false
```

Подсказка: пока никаких стеков не нужно.

# 7. Мини-язык ⭐⭐⭐⭐⭐

Напиши функцию

```
calculate(text string) int
```

На вход приходит строка

```
10+25-7+100-2
```

Функция должна вернуть

```
126
```

Гарантируется, что строка всегда корректная.

Нельзя использовать `strconv.Atoi`.

---

# 8. Самое длинное повторение ⭐⭐⭐⭐⭐

Напиши функцию

```
longestRepeat(text string) (rune, int)
```

Например

```
aaabbccccdd

↓

'c', 4
```

# 9. Мини-пароль 2.0 ⭐⭐⭐⭐⭐

Напиши

```
validate(password string) bool
```

Пароль считается корректным если

- минимум 10 символов
- минимум 2 цифры
- минимум 2 большие буквы
- минимум 2 маленькие буквы
- нет двух одинаковых символов подряд

---

# 10. Мой фаворит ⭐⭐⭐⭐⭐⭐

Напиши функцию

```
normalize(text string) string
```

Она должна

1. убрать все лишние пробелы;
2. каждое слово сделать с большой буквы;
3. остальные буквы сделать маленькими.

Например

```
"   hELLo     woRLD     GO  "

↓

"Hello World Go"
```

Нельзя использовать пакет `strings`.
